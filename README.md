# swarm

Example Usage

Python 2.7

Spin up 1 logger:
python logger.py -s [server IP]

spin up [N] workers:
python worker.py -s [server IP]

spin UP server:
time python server.py -j dns -w [N] --dns-server 192.168.142.150 --dns-query isi150.lappy.lab  --dns-count 100
